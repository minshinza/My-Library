နမော တဿ ဘဂဝတော အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ



နာမ်ဂိုဏ်းနိဒါန်း


နာမ်ရုပ်ကြေမွမှ ဂါထာစုဏ္ဏိယ အစီ အကုံး အနှုန်း အဖွဲ့မြန်တက်, ကျမ်းဂန်တို့၌ အနက်ထင်မြင် သဘောဉာဏ် သက်ဝင်စူးရှနိုင်သောကြောင့် နာမ်ရုပ်ကြေမွအောင် "မနောဂိုဏ်းစု, ပု-စိတ်-ကညာ, ဂု-ဂစ္ဆာနှင့်, ပုမာ-ရာဇ, သတ္ထ-ရတ္တာ, နဒါ-ဂဟံ, သဗ္ဗနာမ်, မှတ်ရန်တစ်ဆယ့်သုံး-ဟူသော တစ်ဆယ့်သုံးဂိုဏ်းတို့၏ ပဒမာလာကို အာလုပ်နှင့်တကွ ဝိဘတ် ၁၆-လုံးတို့ကို အစဥ်အတိုင်း အကျဥ်းမျှ ပြဆိုပေအံ့။ ထိုတွင် မနောဂိုဏ်းစုကို ပုရိသ စိတ္တ ကညာဒိဂိုဏ်းတို့ထက် လက္ခဏာပိုလွန်သော ဂိုဏ်းထူးဖြစ်၍ ထိုသုံးဂိုဏ်းတို့၏နောက်မှ ပြဆိုပေအံ့။"



[ ဂါထာ ]

၈-လုံးလျှင် ၁-ပိုဒ်၊ ၄-ပိုဒ်လျှင် ၁-ဂါထာ စသည်ဖြင့် ပိုင်းခြား၍ ဆန်းကျမ်း၌လာသော အစီအစဥ်အတိုင်း ရေးသားစီကုံးအပ်သည့်အတွက် ရွတ်ဆိုသင့် ရွတ်ဆိုထိုက် ရွတ်ဆို၍ချောမောလှသော...

အသေဝနာ စ ဗာလာနံ၊ ပဏ္ဍိတာနဉ္စ သေဝနာ။
ပူဇာ စ ပူဇနီယာနံ၊ ဧတံ မင်္ဂလ-မုတ္တမံ။

စသည်ကို ဂါထာ-ဟုခေါ်သည်။
ဂါယိတဗ္ဗာတိ ဂါထာ - ဂါယိတဗ္ဗာ-ရွတ်ဆိုအပ် ရွတ်ဆိုထိုက်၏၊ ဣတိ တသ္မာ-ထို့ကြောင့်၊ ဂါထာ-ဂါထာမည်၏။



[ စုဏ္ဏိယ ]

ဂါထာ-ကဲ့သို့ အက္ခရာကန့်သတ်ချက်, အပိုဒ်ကန့်သတ်ချက်မရှိသော ပါဠိစကားပြင်ဟူသမျှကို ယခုကာလ၌ စုဏ္ဏိယ-ဟုခေါ်ကြသည်။



[ နာမ်ဂိုဏ်း ]

ပါဠိလို နာမဂဏ-ကို မြန်မာလို နာမ်ဂိုဏ်း-ဟုခေါ်၏။ [နာမ-နာမ်ပုဒ် + ဂဏ-အပေါင်း]
စည်းကမ်းဥပဒေသချင်းတူသော ပုဂ္ဂိုလ်အပေါင်းကို တစ်ဂိုဏ်းဖွဲ့ရသကဲ့သို့ ရုပ်တွက်ရာ၌ အစီအရင်ချင်းတူသော နာမ်ပုဒ်အပေါင်းကို တစ်ဂိုဏ်းဖွဲ့ရလေသည်။ ရုပ်စီရင်ပုံ ကွဲပြားသလောက် ဂိုဏ်းခွဲ၍ပြလိုက်ရာ နာမ်ဂိုဏ်း ၁၃-ဂိုဏ်းထွက်လာလေသည်။
[ဂိုဏ်းဥပဒေသထက် ထူးခြား၍ကျင့်သော အချို့ပုဂ္ဂိုလ်များရှိတတ်သကဲ့သို့ ဂိုဏ်းစည်းကမ်းထက် ထူးပြားနေသော အချို့ရုပ်များလည်း ရှိကြသည်ကို တွေ့ကြပါလိမ့်မည်။]



[ ပုရိသာဒိဂိုဏ်း ]

ယခုကာလ ထိုထိုဂိုဏ်းဂဏတို့၌ ခေါင်းဆောင်ပုဂ္ဂိုလ်တစ်ဦးစီရှိ၏။ ပုရိသာဒိဂိုဏ်း၌ ပုရိသ-ဟူသော နာမ်ပုဒ်သည် ခေါင်းဆောင်ရုပ်ပင်တည်း။
ပုရိသာဒိ-၌ အာဒိ-သဒ္ဒါသည် အတူ-ဟူသောအနက်ကိုဟော၏။ ထို့ကြောင့် ရုပ်စဥ်ရပုံအားဖြင့် ပုရိသနှင့်တူသော နာမ်ပုဒ်အပေါင်းကို ပုရိသာဒိဂိုဏ်း-ဟုခေါ်၏။ [ပုရိသာဒိဂိုဏ်း-ပုရိသနှင့်တူသော နာမ်ပုဒ်တို့၏အပေါင်း]



[ ပဒမာလာ ]

[ပဒ-ပုဒ်၊ မာလာ-အစဥ်အတန်း]ပုဒ်တစ်ခုတစ်ခုကို ဧကဝုစ် ဗဟုဝုစ် အစဥ်ကျအောင် ပထမာ ဒုတိယာ-စသော ဝိဘတ်အစဥ်ကျအောင် စဥ်တန်းမူကို ပဒမာလာ-ဟုခေါ်၏။
သဘောတူ နာမ်ပုဒ်အပေါင်းကို ဂိုဏ်း-ဟုခေါ်၏။ ထိုထိုဂိုဏ်းဝင် နာမ်ပုဒ်တစ်ခုတစ်ခုကို ပုရိသော ပုရိသာ-စသည်ဖြင့် စဥ်ခြင်းကို ပဒမာလာ-ဟုခေါ်သည်။



[ ကာရန္တ ]

[ကာရ + အန္တ ကာရ-အက္ခရာ + အန္တ-အဆုံးရှိသည်]
တည်သောအခါ၌လည်းကောင်း ပြီးသောအခါ၌လည်းကောင်း အ-အာ-ဣ-ဤ-ဥ-ဥု-သြ-အံ(ဗိန္ဓု - နိဂ္ဂဟိတန္တ) ဟု အက္ခရာ-၈-လုံးတို့တွင် တစ်လုံးလုံး ဆုံးလျက်ရှိသည်ကို ကာရန္တ-ဟုခေါ်သည်။



[ ပကတိ ]

တစ်စုံတစ်ခု ဝိဘတ်မသက်ခင် ပင်ကိုပုဒ်အတိုင်းကို ပကတိ-ဟုခေါ်၏။ ထို့ကြောင့် ပုရိသ-သည် အကာရန္တ-ပကတိရှိ၏။ ပင်ကိုပုဒ်အတိုင်းမှာ သ-၌ အ-အက္ခရာအဆုံးရှိသည်-ဟူလို။



[ လိင်္ဂ ]

လိင်-ကို ပါဠိလို လိင်္ဂ-ဟုခေါ်၏။ လိင်္ဂ-သဒ္ဒါသည် အမှတ်အသား-ဟူသော အနက်ကိုဟော၏။ သဒ္ဒါဆရာတို့က ရုပ်စီရင်ရာ၌ သန့်ရှင်းလျှင် ပုလ္လိင်-ဟုခေါ်ကြသည်။ ထို့ကြောင့် မိန်းမ-ဟူသော အနက်ကိုဟောသော မာတုဂါမော, သြရောဓော, ဒါရော-တို့သည် ပုလ္လိင်သာတည်း။

ရုပ်စီရင်ရာ၌ မသန့်ရှင်းသော ရှူတ်ထွေးသော ပုဒ်များကို ဣတ္ထိလိင်-ဟုသတ်မှတ်ကြသည်။

ရုပ်စီရင်ရာ၌ သန့်ရှင်းသည်လည်းမဟုတ်၊ မသန့်ရှင်းသည်လည်းမဟုတ်သော နှစ်မျိုးလုံးရောထွေးလျက်ရှိသော ပုဒ်များကို နပုလိင်-ဟုခေါ်ကြလေသည်။



[ နာမ်ရုပ်တို့၌ ကျွမ်းကျင်ပါလျှင် ]

နာမ်ရုပ်တို့၌ ကျွမ်းကျင်ပါလျှင် ဂါထာ စုဏ္ဏိယ-များ ရေးသားစီကုံး သုံးနူန်းဖွဲ့ဆိုရာ၌ မှန်ကန် သွက်လက် မြန်ဆန်ပါသည်။ ပါဠိတော် အဋ္ဌကထာ ဋီကာကျမ်းဂန်တို့ကို ကြည့်ရှုရာ၌လည်း အနက်အဓိပ္ပါယ်ကို တိကျစွာ သက်ဝင်စူးရှ သိနိုင်ပေသည်။



[ နာမ်ဂိုဏ်း ၁၃-ဂိုဏ်း ]

သဒ္ဒါပိုင်လိုလျှင် နာမ်ကိုနိုင်ရမည်-ဆိုသည့်အတိုင်း နာမ်ဂိုဏ်း ၁၃-ဂိုဏ်းတို့ကို သဘောပေါက်နားလည်ရန် အထူးပင်လိုအပ်လှပါသည်။ စီရင်ထားသော နာမ်-ရုပ်စဥ်များနှင့် ထူးခြားသော နာမ်-ရုပ်စဥ်များကို စနစ်တကျလေ့လာနိုင်ရန် ထိုနာမ်ဂိုဏ်း ၁၃-ဂိုဏ်းတို့၏ ပဒမာလာကို အာလုပ်ဝိဘတ်နှင့်တကွ ဝိဘတ် ၁၆-လုံးတို့ကို အစဥ်အတိုင်းပြဆိုရပေလိမ့်မည်။

    ထိုနာမ်ဂိုဏ်း ၁၃-ဂိုဏ်းတို့မှာ -

  • ၁။ ပုရိသာဒိဂိုဏ်း
  • ၂။ စိတ္တာဒိဂိုဏ်း
  • ၃။ ကညာဒိဂိုဏ်း
  • ၄။ မနောဂိုဏ်း, မနောဂဏာဒိဂိုဏ်း
  • ၅။ ဂုဏဝါဒိဂိုဏ်း
  • ၆။ ဂစ္ဆန္တာဒိဂိုဏ်း
  • ၇။ ပုမာဒိဂိုဏ်း
  • ၈။ ရာဇာဒိဂိုဏ်း
  • ၉။ သတ္ထာဒိဂိုဏ်း
  • ၁၀။ ရတ္တာဒိဂိုဏ်း
  • ၁၁။ နဒါဒိဂိုဏ်း
  • ၁၂။ ဂဟပတာဒိဂိုဏ်း
  • ၁၃။ သဗ္ဗနာမ်ဂိုဏ်း

ပုရိသာဒိဂိုဏ်း

တည်သောအခါ ပုရိသ-ဟု အ-ကာရန္တဖြင့် တည်ပြီးလျှင် သော-ဟူသောသုတ်ဖြင့် သိ-ဝိဘတ်ကို အော-ပြု၍ ပြီးသောအခါ ပုရိသော-ဟု သြ (အော)-ကာရန္တဖြင့် ပြီးကောင်းသောပုဒ်အပေါင်းသည် ပုရိသာဒိဂိုဏ်းမည်၏။ ဤပုရိသာဒိဂိုဏ်းသည် ပုလ္လိင်-သာတည်း။ ရုပ်စဥ်ဟန်ကား-
